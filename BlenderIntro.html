LESSON #2: Editing an Object
Moving primitives to build shapes is good for roughing out a space, but detailed modeling means Cutting, Extruding, and Transforming Vertices, Edges, and Faces to get the precise form and details you intend.


  [2.1] START with a new file.

  1. Delete the default cube.
  2. Create a new Cube: Add > Mesh > Cube.


  [2.2] EDIT MODE:

Select the Cube, Hit [Tab] for [Edit Mode].
(or in UpperLeft corner switch [Object Mode] to [Edit Mode]).

This allows component mode selection (vertices, edges, and faces), Transforms of those components, and other editing (like cutting and extruding).


  [2.3] COMPONENT TYPE

Choose Vertex, Edge, or Face Components by clicking icons next to [Edit Mode] in the upper left corner.
Or use keyboard shortcuts: Vertex [1], Edge [2], Face [3].

  • Select part of the mesh: Choose a component type, hit [w], and LeftClick the object part to select it. Hold [Shift] to add to the selection. Note the select modes in the top bar, which you can toggle through by hitting [w] repeatedly.

  • Transform the selection: Use Move [g], Rotate [r], or Scale [s] to transform it ([x], [y], [z], to choose a direction).

  • Expand/Shrink Selection: Select a face, [Ctrl/Cmd]+[+] to expand how much is selected, [Ctrl/Cmd]+[-] to shrink selection.

  • Select a loop: In Edge or Face mode, select one edge or face, hold [Alt / Option] and select a neighbor to select an entire loop.

  • Soft-Select ("proportional editing"): Select a component, like a vertex. Hit [o] to transform the surrounding geometry according to a threshold. Hit [o] to return to normal selection.

  • Select through the model: By default, selecting components around a mesh only effects the Faces, Edges, or Vertices facing the view. The back side is not effected ("back face culling").
    To select through the model enter wirefarme or X-ray mode.



  [2.4] GEOMETRY TOOLS

The following are tools for cutting and extruding geometry to create new forms.
For more tools (like the boolean modifier) see Lesson 5: Character Modeling

1. LEFT-COLUMN TOOLS:


  • [Extrude] / [e]: Select a face, hit Extrude to pull out new geometry on the face "normal".

  • [Inset] / [i]: Create a new "inward extrusion" on the model surface.

  • [Bevel] / [Ctrl]+[b]: Select a corner edge, hit bevel to make rounder (splits and separates into more edges). Use the tool menu (LowerLeft) to choose the number of Segments (divisions).

  • [Knife] / [k]: Hit Knife to click on edges, vertices, or faces to cut new lines into the geometry. These can then be Transformed or Extruded

  • [Loop Cut] / [Ctrl]+[r]: Hit Loop Cut to cut a smooth Edge Loop line around the entire form. In the tool menu (lower-left) change the number of segments, and then non-uniform Scale to set distance.

Hold down the Edge Loop button to switch it: Offset Edge Loop Cut. With Offset Edge Loop Cut activated, LeftClick and drag on a previously cut Edge Loop to spread-cut new edge loops around the first.

Draw a line between two selected vertices (called "connect" in other programs): select the vertices, hit [f]

  • [Edge Slide]: Select an Edge to drag it along the existing space. Use after Edge Loop to adjust position.



    2. DELETE OPTIONS:

    Select a Face or Edge and hit [Delete] or [x] for delete options:

    • Make a hole: Select a Face, hit [Delete] / [x], choose "Face".

    • Dissolve an Edge: Made an Edge you do not want?
      Select it (or the entire loop with [alt]+select neighbor), hit the [Delete] or [x] key, choose "Dissolve Edge".


    3. RIGHT-CLICK FUNCTIONS : (or Modeling menu > Mesh):


  • Bridge Edge Loops: In Edge mode, select opposite hole edges, RightClick to choose "Bridge Edge Loops" to make geometry across the gap.

  • Bridge Faces: Select two opposing faces, use Inset to pull geometry inward, RightClick to choose "Bridge Faces" to punch a window between them.

  • Merge [m] as Edge collapse (turn an Edge into a Vertex): Select an Edge, RightClick and choose Merge to collapse it.

    Merge [m] as Vertex target weld (snap one vertex to another and merge them): Select two or more vertices, hit [m] and from the menu choose "At First" or "At Last" to snap /merge the vertices to the first or last selected.

  • Shade Smooth / Shade Flat (edge smoothing / hardening):
    To make an Edge between Faces appear smooth, select the Faces, RightClick and choose "Shade Smooth" (Or Modeling menu > Mesh > Normals > Shade Smooth). This has the benefit of showing you prblematic geometry!

    To make an Edge between Faces appear sharp, select the Faces, RightClick and choose "Shade Flat" (Or Modeling menu > Mesh > Normals > Flat Smooth).




      [2.5] INTERFACE TOOLS

    USE X-RAY MODE FOR SEEING AND SELECTING:
    1. Find the X-Ray Mode button in the viewport UpperRight, a Double-square icon before the Viewport Shading options.
    2. Turn this on to see through the model, and backfaced components can be selected / Transformed along with front-facing components.

    SCALE WITH INDIVIDUAL ORIGINS
    By default, scaling multiple faces will also scale the distance between them:
    1. Divide up a mesh and select non-adjacent Faces.
    2. Hit [s] and drag to Scale. The distance between the faces scales along with the faces themselves.

    To not change the distance when Scaling (keep distance consistant), we need to turn on "Individual Origins".
    1. In the top bar, next to Transform Orientations (Global or Local), find the "Transform Pivot Point" options.
    2. Switch from "Bounding Box Center" to "Individual Origins".
      Now scale the faces, and the distance between them is unchanged.

    LESSON #3: Intro to 3D Object Animation:
    Every motion in 3D Animation requires at least two keyfames: an initial keyframe with the start value and an end keyframe with the changed value. Blender interpolates the motion between those keyframes.

    To vew the Timeline, choose the "Layout" option in the top menu (Timeline should be at the bottom). To vew the Graph Editor instead of the basic Timeline, we switch the panel contents (click the upper left button in the panel).

      [3.1] ANIMATION CONCEPTS:


    The Timeline is a linear visualization of all animation for selected objects. It is incremented as frames, 30 frames per second. Set the active frame range length in the UpperRight corner.


    A Keyframe contains time and value information for a specific object parameter, such as Position on the Z axis.

    Every movement takes at least two Keyframes: an initial value at one frame, and the changed value at a later frame.

    On the Blender Timeline, a Keyframe is a white diamond (yellow if selected). A single Keyframe in the Timeline may actually be multiple keyed parameters stacked on the same frame.

    If the keyframe is dragged to a new frame, the animation timing changes. Moving two keyframes closer together creates faster motion (less time to cover the same change in value). Motion is more interesting and expressive if the timing varies.

    "Bookends" in 3D animation are two identical keyframes at different times. Changes between them do not effect the timeline outside of them, and changes outside of them do not effect the timeline between them.

    An "Animated Hold" is two keyframes that are almost but not exactly identical, to show a pause-in-motion. This can indicate character attention and thought, and helps prevent a rubbery "constant motion" appearance in your animation.

    By default, most movements include ease-in and ease-out (slow at start, pick up speed, slow at end). This is seen in the Graph Editor as an S-curve, and in cases where a more constant motion is preferred, the keyframe bezier handles can be adjusted to make a straight diagonal line.

    The 12 principles of Disney Animation apply as much to 3D work as they do to 2D. Consider Squash and Stretch to show flexibility, force, and impact, moving in arcs to create organic motion, and anticipation to focus viewer attention on the coming action (as well as to show character intention and flexibliy). All of these ideas should be considered when animating to create expressive, impactful results.

    Add depth and a sense of tangibility to your model with an Ambient Occlusion pass.
    AO refers to how a model self-shadows.
    For example consider the shadows where clothing wrinkles, overlaps other clothing, or overlaps skin.


    STEP 4b.0: Setup the workspace:

    Split the main viewport to show UV Map in one panel. With your object selected hit [tab] for edit mode, to view the UVs. Note the "UV Sync Selection" button in the upper-left to select faces in the model.    

    Ideally, start with an unwrapped model:
    An AO render uses existing UV maps in the object to place textures.

    Once the AO bake is set up, it is pretty simple to revise the UV map and re-render the AO, but ideally the UV map is complete for this tutorial.



    To test this process without a finished unwrap, try a fast Auto UV Unwrap:

    1. Select the object, Apply Scale: Object > Apply > Scale.
    2. Hit [tab] for [Edit Mode], hit [2] (Edge Select), hit [a] (select all).
    3. Hit [u] for UV Unwrap menu > Smart UV Project.
      Set "Island Margin" gap. Try 0.05 (keep under 0.1).
      Hit [ OK ] to see the object UV Unwrapped (all fits in UV square).


    STEP 4b.1:
    Create an Image to bake onto:
    In the UV Editor click the New Image button (top center).
    • Name = ObjectName_AO
    • Width = 2048
    • Height = 2048
    • Click color, set Alpha = 0, so background is blank
    • Hit [ OK ].




    STEP 4b.2: Add this Image to the Object Material :


    1. If the Object does not yet have a Material, add a Material:

    1. Select the object. In Properties > Material tab, hit [ new ].
    2. Set shading in the viewport to show materials.







    2. Add an Image Texture to the object Material's Shader Graph:

    1. In the bottom panel's top menu, switch interface from the UV Editor to the Shader Editor.
    2. Make sure this is turned off: find File menu > External Data> Automatically Pack into .blend.
    3. With the mouse over the Shader Graph, hit [Shift]+[a] to search and create an Image texture.
    4. Click the Mountain Icon on the left (in the Image Texture, next to [+ new]) to choose the Image (ObjectName_AO).
    NOTE 1: The AO texture does not get connected to the Material. It just lives in the same material space in the Shader Editor.

    NOTE 2: if your object has multiple materials, you can copy and paste this Image Texture node into the shader graph of each material: Select the Image Texture node, [Ctrl/Cmd]+[c] to copy, open another material the Shader Graph, to [Ctrl/Cmd]+[v] paste (make sure your cursor is hovering over the Shader Graph to paste in there).

    Save your work so far!


    STEP 4b.3: Time to bake the AO texture:
    In the bottom panel's top menu, switch back to the UV Editing tab.

    Open Project panel > Render Properties tab (second from top) :
    1. Set Render Engine = Cycles, Sampling > Render = 32 (a low number is good for a quick test, higher numbers gets better final resulst but can take multiple minutes to process).
    2. Open the Bake rolldown, change Bake Type from Combined to Ambient Occlusion. Set Margin = 4 px.
    3. Hit the [ Bake ] button. View the results in the UV Edit panel.
    4. For higher-quality results (to remove noise):
      Set Sampling > Render = 512 and hit [ Bake ] again.






    STEP 4b.4: Save the AO texture to bring into Photoshop:
    In the Blender UV Editing tab, select all faces by hitting, in order:
      1. [tab] (Edit Mode)
      2. [3] (Face Mode)
      3. [a] (Select All)
    (or, enter X-Ray mode, and select around the entire mesh)




    In Blender's Project panel > Render Properties tab.
    1. Set bottom Color Management > View Transform from Filmic to Standard (for a simple gray background)

    2. In the UV Edit panel, Image > Save.
      This saves the ObjectName_AO image (that we created and baked) as a PNG.


    In Photoshop we combine the color and AO files:
    1. Open the new AO file: File > Open

    2. Copy and paste the content into your current texture painting PSD:
      [Ctrl/Cmd]+[c] and [Ctrl/Cmd]+[Shift]+[v]

    3. In Layer Manager: name it AO_layer, move it above painted art.

    4. Set the Blending Mode on AO_layer = Multiply.
      This AO layer can now be duplicated if you want a darker result [ Ctrl / Cmd ]+[ j ], or lower its Opacity to make it lighter, as desired.

    5. File > Export > Quick Export as PNG named "ObjectName_colors.PNG"
      (older Photoshop: File > Save As).




    Back in Blender:

    1. Switch the bottom panel to the Shader Editer.
    2. With the mouse over the Shader Graph, hit [Shift]+[a] to search and create an Image texture.
    3. Hit the Open Image icon to load the ObjectName_colors.PNG.
    4. Drag a connection line from the "Color" circle to the "BaseColor" circle on the Principled BSDF material.

    In the 3D Scene view upper right, turn on Viewport Shading to display the textures on the model .

      [5.1] ADD A REFERENCE IMAGE:

    To lock the image object so it cannot be selected:

    1. In the Outliner, click the Filter (beaker) icon in the upper-right.
    2. At the top, under "Restriction Toggles", click the selection arrow (second icon form the left).
    3. A column of selection arrows now appears in the Outliner.
    4. Turn off the arrow for the reference image.


      [5.2] USE X-RAY MODE FOR SEEING AND SELECTING:

    1. Create a cylinder: Add > Mesh > Cylinder, so there is something to see through.
    2. Find the X-Ray Mode button in the viewport UpperRight, a Double-square icon before the Viewport Shading options.
      Or hit [Alt / Option]+[z]
    3. Turn this on to see through the model, and select through the model: backfaced components can be selected / Transformed along with front-facing components.


      [5.3] MIRROR GEOMETRY:

    In Blender, Instanced Mirroring is a Modifier.
    1. Select Mesh to be mirrored (like half a body).

    2. In [Object Mode], go to the Properties panel, open the Modifiers tab (wrench icon).

    3. From the rolldown choose Mirror.

    When modeling is done, delete the Mirror modifier.
    Duplicate the mesh, use the [Object] Mirror function to reverse the duplicate on X Global.


      [5.4] JOIN (COMBINE) GEOMETRY:
    1. Select two Meshes in [Object Mode].

    2. Hit [Ctrl / Cmd] + [ j ] to join (or open the [Object] menu and choose Join).
      The two Meshes are now one mesh.

    3. To connect the forms at their vertices, [Delete] any interior polygons, use Knife to cut any needed matching vertices, select pairs of matching vertices and hit [m] to merge (at center).


      [5.5] BOOLEAN GEOMETRY:

    In Blender, Boolean operations is a Modifier.
    Boolean operations including turning two meshes into one, or cutting one mesh out of another.

    Given two intersecting meshes:
    1. Select a Mesh. It is the Primary.

    2. In Object Mode, go to the Properties panel, open the Modifiers tab (wrench icon).
      From the rolldown choose Boolean.

    3. Click the eyedropper and select the overlapping mesh. The default operation is Difference, so the new mesh cuts a shape out of the Primary mesh.

    4. Move the new mesh to see the cut shape dynamically change.

    5. To apply the change, hit [Ctrl]+[a] and choose "Visual Geometry to Mesh" (or click the rolldown next to Boolean and choose "Apply").

    Also try Union and Intersection operations.


      [5.6] FREEZE TRANSFORMS:

    Select a Mesh and open the Object menu.
    Apply > All Transforms.

    NOTE: Blender does not maintain individual object history, so there is no concept of "Delete History" as there is in Maya.

      EXERCISE #5: Character Rough Modeling:

    Reference   |   Torso   |   Limbs   |   Foot / Hand   |   Head   |   Mirror   |   Next

    The following character process is called "Volume Modeling."
    It allows us to create quick forms for testing proportions and staging in a film or game pipeline.

    Further Steps would be to cut and extrude clothing and face details.



      STEP 5.1: Prep: Reference Diagram and X-Ray:

    (Click images for full images)

    PHOTOSHOP
    Create a reference image (PNG, save in production folder).
    Draw front and side views, and optionally a back view. Try to keep all images in proportion to each other.


    BLENDER
    1. In Cartesian Navigator (UpperRight), click for front view.

    2. In Object mode, hover mouse over viewport.
      1. Hit [Shift]+[a].
      2. Choose Image > Background to load the image from your production folder.

    3. Select the imported art, hit the Move tool [g] and move it up and to the side until the feet stand on the ground and the front view is centered at world origin (0).

    4. Turn on X-Ray mode: hit the Double-square icon in the upper-right ([Alt / Option]+[z]). The default Cube should now be see-through.

    5. Delete the default cube.



    BLENDER OUTLINER
    1. In the Outliner panel (upper-right), rename the empty layer "Reference".

    2. Lock the reference layer: 
      1. In the Outliner upper-right hit the Filter button to activate the "Selectable" filter (triangle icon).
      2. A column of triangles now appears in the Outliner panel.
      3. Click on the arrow in the same layer as Reference to lock it (so it cannot be accidentally selected while modeling).
    The Reference image will only be visible in the Front view, so when we MiddleMouse click-drag to orbit the view, the image seems to disappear. Just hit on the UpperRight Gizmo to return to the front view and see the reference.
    To pan the view: [Shift]+MiddleMouse click-drag.


      STEP 5.2: Shape the TORSO:

    1. Create a Cylinder: Add > Mesh > Cylinder.

    2. In the LowerLeft panel (available when first adding a mesh), set Vertices (divisions) = 8. If you miss the panel (accidentally performed another action), delete the Cylinder and Add again.

    3. In the Front view, Move the Cylinder up and non-uniform scale vertically to match the diagram torso height, neck to crotch.


    4. Enter Edit Mode [Tab].

        Use LoopCut and EdgeSlide to cut and move 5 lines:
        Shoulders, Armpits, Chest, Waist, and Hips.


    5. For each of the 7 Edge Loops (including top and bottom):
      1. Select an Edge loop: select 1 edge, hold [Alt], select next.
      2. OR select a row of vertices: in Vertex mode and Select Box mode, select around the row.
      3. Hit the Move tool to drag vetically (or hit [g] and [z]).
      4. Hit [s] to scale so the silhouette matches the diagram.
      5. Scale smaller the Cylinder top and bottom (neck and crotch).


    6. Orbit (middle mouse click and drag) to view the bottom of the model (an octagon).

    7. Use the Knife tool to cut the top and bottom:
      How to use Knife: Click the Knife tool on left, click one vertex, click the opposite vertex, hit [Enter] to complete the cut.
      1. Cut two lines on the bottom: down the middle front to back, and then across it side to side. Select the center vertex and drag it down a little bit.
      2. Repeat cuts for top octagon: Orbit, Knife cut middle lines.

    8. In the Front view, in Face mode, select all left-side faces.

    9. Hit [Delete] and choose Face to remove that side.



    Create an Instanced Copy for the Profile view:
    1. Make an instanced copy:
      1. Hit [tab] to exit Edit Mode and return to Object Mode.
      2. Hit Object > Duplicate Linked (or [Alt / Option] + [d]) to make a copy that is actively moving, then hit [x] to move it sideways to cover the profile character drawing.

    2. With the duplicate mesh selected, under Object Properties rotate the model 90 degrees on the Z axis to cover the art.

    3. In Edit Mode [tab] move the duplicate mesh Vertices to follow the character profile silhouette.



      STEP 5.3: Extrude LIMBS:

    The arm and leg will be extruded from hexagons on the side.

    ARM:
    1. Move vertices around the shoulder area to form a hexagon between the shoulder and armpit loops (2nd and 3rd horizontal lines from the top).

    2. Select the edge within the hexagon, hit [Delete] and choose Dissolve Edge to remove that edge.

    3. Switch meshes: Hit [tab] to exit Edit mode, select the original mesh, hit [tab] to enter Edit mode on the original mesh.

    4. Extrude the arm:
      1. In Face mode select the shoulder hexagon (may need to orbit to access this side of the mesh).
      2. Back in the Front view : Hit [e] to Extrude and move out sideways to the wrist.


    5. Scale the end:
      1. Shrink: Hit [s] to scale smaller, for the wrist-width.
      2. Flatten: Hit Scale tool to non-uniformly scale the red X axis.


    6. Use LoopCut to cut an edge across the middle of the arm at elbow-length.

    7. In Top view move this edge loop back for the elbow bend.
      Hit [tab] to exit Edit mode.


    LEG:
    a) Re-select the side mesh, hit [tab] for Edit mode.
    1. Choose Edge mode.
    2. Find and select the side-center vertical Edge between hips and crotch loops.

    b) Scale [s] that line slightly bigger.

    c) Hit [Delete] and choose Dissolve Edge.


    d) Switch meshes: Hit [tab] to exit Edit mode, select the original mesh, hit [tab] to enter Edit mode on the original mesh.


    e) Extrude the leg:
    1. In Face mode select the leg hexagon (may need to orbit to access this side of the mesh).
    2. Back in the Front view : Hit [e] to Extrude out.
    3. Move down to the ankle location.


    f) Scale the end:
    1. Shrink: Hit [s] to scale smaller, for the ankle-width.
    2. Flatten: Hit Scale tool to non-uniformly scale the blue Z axis.

    g) Use LoopCut to cut an edge across the middle of the leg.


    h) [tab]-select-[tab] to switch back to the side-mesh Edit mode. Move the knee edge loop "forward" to match the profile bend of the knee.



      STEP 5.4: Extrude FOOT and HAND:

    FOOT:
    1. With the side mesh selected, in Edit Mode and Face mode, orbit to view and select the ankle hexagon.

    2. In Front view : hit [e] to Extrude this face down for the foot height.

    3. Select the 3 faces on this Extrusion that face character front.

    4. Hit [e] to Extrude forward for the foot length.

    5. In Side view select / move toe vertices for tapered end.


    HAND:
    Extrude the Wrist and Palm
    1. With the front mesh selected, in Edit mode and Face mode, select the wrist hexagon.

    2. Hit [e] to Extrude this face out for the wrist. LeftClick to end.
      1. Hit [s] and scale larger.
      2. In the Top view , non-uniform scale even wider.


    3. In Front view : hit [e] to Extrude the palm. LeftClick to end.

    4. Hit [e] to Extrude the space between thumb and fingers. LeftClick to end.

    5. Click the Scale tool, non-uniform Scale the end shorter (vertically).

    6. Center the selection in the viewport: hit [~] to choose "View Selected".


    Create Base Faces for Fingers
    1. Orbit to view the hand at an angle.

    2. Turn off the Toggle X-Ray mode.

    3. Use the Knife tool to cut 3 vertical lines on the end face.

    4. Select the lines and use Bevel Edge to split into pairs, for the spaces between fingers.

    5. Move vertices (and non-uniform Scale edges) to make the four finger-faces more square and even.

    6. In Face mode select and move each square to form a curve from the side and and from the top.



    Extrude and Shape the Fingers
    1. Select the four finger Faces, hit [e] to Extrude the fingers.

    2. Go to the Top view . Turn on X-Ray mode again, so that selections can capture back-facing vertices.

    3. Shape finger lengths
      1. In Vertex mode, select around a finger end (will select all four vertices if X-Ray mode is active).
      2. Use Move tool to lengthen and change the finger angle.
      3. Hit [s] to Scale this finger-end to decide tapering.
      4. Select, move, and scale the other finger-ends to adjust lengths, spread, and tapering.

      NOTE: Alternatively, to scale multiple finger-ends at the same time: In the top bar, set "Transform Pivot Point" to "Individual Origins", select all the finger ends faces, hit [s] to scale all finger tips a bit smaller.

    4. Create Joints: Use LoopCut and EdgeSlide to cut and move 2 lines for the finger joints.

    5. Select / move these lines a bit up, to gently curve the fingers.

    6. Similarly, Extrude a thumb from the side (facing front):
      1. Select the face.

      2. Hit [e] to Extrude. Move out slightly for a base. Click to end.

      3. Hit [s] to Scale a bit smaller. Move tool to drag a bit down.

      4. Hit [e] to Extrude the thumb length. Click to end.

      5. Hit [s] to Scale a bit for a tapered end.

      6. Use Loop Cut to slice a line across the thumb middle

      7. Select the end joint and Move it down for a bent thumb.


    7. Cut lines in the hand to split any polygons over 4-sides:
      Complete the top of the hand:
      1. In Vertex mode, use Knife to draw triangles from one side of the finger bases back to the wrist.
      2. Cut triangles from the other finger base sides back to the next loop.
      3. Select Edges to adjust the back of the hand a bit flatter.

        Orbit to view under the hand to complete the palm:
      4. In Vertex mode, use Knife to draw triangles from the finger bases back to the wrist.
      5. Cut triangles from the other finger base sides back to the next loop.
      6. Select Edges to adjust the palm: add the primary dent in the center, make the two sides more padded.

        Hit [tab] to exit Edit Mode and return to Object mode.

      STEP 5.5: Build the Head:


    a) In Object mode, create a 2D Plane: Add > Mesh > Plane.
    1. In Object Properties Rotate X = 90.
    2. Move to the top of the profile drawing neck.
    3. Scale it to a bit larger than the intended head and neck.

    b) In the Front view , with the Plane selected, hit [tab] turn on Edit mode.
    1. Use the Knife tool to cut the head and neck profile into this plane, ending on the first cut.
    2. Cut 2 vertices for each feature: 2 for the chin, 2 for each lip, the nose tip, etc.
    3. Hit [Enter] to complete the cut.


    c) In Face mode select and [Delete] faces surrounding the profile.
    1. Hit [tab] for Object mode, create an instance copy of the head shape: Object > Duplicate Linked.
    2. Move to the front view, Rotate Z = -90.
    3. Move to position in line with the body center.


    d) Hit [tab] for Edit mode. Make 3 Extrusions to create volume:

    1a. A thin Extrusion for the width of half the nose.
    1. Select the edge loop [Alt]
    2. Bevel slightly OR, Scale the face Uniformly
    3. select the wider edge loop and Delete > Dissolve Edge.


    1b. Use Knife to cut a separation for the nose shape.





    2a. A thicker Extrusion for neck width.
    1. Scale Uniformly: Select Edge loop [Alt] and Bevel a bit.
    2. Select the wider edge loop and Delete > Dissolve Edge.


    2b. In Edge mode select mouth corner Edges and Merge [m].

      Use Knife to separate the neck.




    3. A thinner Extrusion for the head side.

      Scale smaller.




    e) Build an ear with 4 extrusions:
    1. First Extrusion, to make the ear base:
    1. Select the head side.
    2. Hit [e] to Extrude a small amount.
    3. Scale smaller to start the shape of the ear base.
    4. [tab] to exit edit mode, select the profile mesh, [tab] to enter Edit mode.
    5. In the Front view, in Vertex mode, select the ear base vertices to move over the ear drawing (further back in the head) and form the ear base shape.
      Use the Knife tool to cut across the neck, connecting the front to the back.

    2. Second Extrusion, full ear width:
    1. Orbit to perspective view, turn off X-Ray mode.
    2. In Face mode select the ear base.
    3. Hit [e] to Extrude out, for the full size of the ear.
    4. Scale [s] bigger, move back [y], rotate forward [z].


    3. Third Extrusion, to round the ear:
    • [e] to Extrude out a bit. Scale a bit smaller.


    4. Fourth Extrude, for the ear canal:
    1. [e] to Extrude inward. Scale smaller.
    2. Move out edges to make the tragus.
    3. Knife-cut a line from the ear down to the neck bottom, to curve the neck.


    f) Complete the rough head model:
    1. Orbit the view to see the head in perspective, and slightly above.
    1. Note the original extrusion lines are not evenly spaced. The nose line is too close to the head center. We want these to be evenly-spaced for a smoother curve.
    2. In Edit Mode and Edge mode, select an edge on this nose-line.
    3. [alt]+select the next Edge to select the entire loop.
    4. Orbit more to the front. Hold down [Ctrl/Cmd], de-select the Edge right above the nose.
    5. Move the edge loop to create more even spacing between the original extrusion lines.


    2. In Face mode, select the Faces inside the neck and head.
    1. Hit the [Delete] key, choose Delete Faces.
    2. Repeat for deleting the top faces on the torso that are inside the neck.
    NOTE: Want to isolate the head for these deletes?
    Select he head in Object mode. Hit [Shift]+[h] for isolation mode.
    Make the changes, and hit [Alt/Cmd]+[h] to exit isolation mode.
    Or, just hit [ / ] in object mode, both to enter and exit isolation mode.


    g) Join the head to the body and Merge vertices:
    1. MATCH VERTICES: The torso's neck hole has 2 less vertices than the head's neck, 1 in the front and 1 in the back.
    1. With the torso selected, [tab] for Edit mode.     
    2. Use the Edge Loop tool to cut a vertical line down the front and up the back.
    3. Move the new Edge Loop line forward and back to make a more curved torso.


    2. JOIN: Hit [tab] for [Object Mode].
    1. Select the torso and [Shift] to add the head.
    2. Hit [Ctrl / Cmd] + [ j ] to join.



    3. MERGE: Hit [tab] for [Edit Mode].

      Select pairs of vertices and hit [m] to merge (at center).



      STEP 5.6: Mirror and Join into one full mesh:


    a) Mirror:
    1. Make sure all Transforms are "applied" (zero-d out):
    We want the current Location, Rotatation, and Scale, AND all properties to show Location and Rotation values = 0, and Scale values = 1.
    1. Hit [tab] for [Object Mode]. Select the mesh.
    2. Hit Object > Apply > All Transforms

    2. Make the Duplicate:
    1. Hit [Shift]+[d] to copy (or Object > Duplicate Objects) and
      LeftClick to end automatic movement, so it stays in place
      (if it moved, type Locaition values in Object Properties = zero).
    2. In Properties panel > Object Properties tab, set Scale X = -1.
    3. Hit [Ctrl/Cmd]+[a] for Apply menu, choose Scale.


    b) Join the two halves into a single mesh and Merge vertices:
    1. Hit [tab] for [Object Mode].
    2. Select the two body halves.
    3. Hit [Ctrl / Cmd] + [ j ] to join (or Object > Join).


    c) Merge Vertices:
    1. Hit [tab] for [Edit Mode]. Turn on X-Ray mode.
    2. In the Front view , select around all center Vertices.
    3. Hit [m] to merge and choose "By Distance".
    4. At Viewport bottom left, open the Merge By Distance menu
      and choose the threshold, to merge intended pairs but not
      unintended neighbors.


    d) Smooth:
    1. Hit [tab] for [Object Mode].
      Select the mesh.

    2. Hit Object > Shade Smooth.

    NOTE: If Shade Smooth results in a strangely shaded mesh, you may have inverted normals. Try this:
    1. Hit [tab] for Edit mode.
    2. Turn on X-Ray mode to allow selecting through the model.
    3. Select all faces, Mesh > Normals > Recalculate Outside.


    e). Move the Pivot to the bottom:
    1. The character should be standing at world center (the "Origin").
    2. With the character selected, move its pivot:
            Object menu > Set Origin > Origin to 3D Cursor
      This allows the character to be scaled larger or smaller without changing its position on the ground.

    That's it-- the low-poly, rough character model is complete!


      WHAT DO I DO NEXT?: Detailed Modelling, and more


    (Click images for bigger, full images:)

    [1]. TEST THE MODEL IN THE PROJECT
    In film or game production, this low-poly, rough model should be tested in the project for correct proportions before adding any more detail!

    Select the character mesh. File > Export as an FBX (in FBX Export settings, turn on "export selected" so only the selected mesh gets exported).
    Import into the project to test.

    Once the mesh is confirmed to be the right sizing for the game or film, continue working on the model, textures, rig, and animation:



    [2]. KEEP MODELING: REVISE MESH DETAIL
    Revise the mesh for the more detailed model.
    1. Delete half of the model again so all work can be mirrored.

    2. Add more Edge Loops across the torso and limbs to get smoother silhouette curves. Every new line should be adjusted (moved, scaled) to contribute to the curves.



    3. Loop Cut to add Edge Loops to the joints (elbow and knee) to support animation.




    4. Loop Cut one long Edge Loop down the arm / hand:
      From the torso-front down the arm-front through the fingers to the arm-back and torso-back.

      Move this line to make curves (move front edges a bit front, back edges a bit etc).

      To curve the fingers, Scale slightly smaller the top and bottom of each finger joint and tip (the edge loop increases finger divisions from 4 divisions to 6).


    5. Add detail to the face: Cut loops around the eyes and from the nose across the cheek, striving for quads (no polygons larger than 4-sided).




     
    [3] ADD HAIR:
    Hair should be a separate, non-symmetrical object.
    a. To make the hair fit the head, extrude from a scalp copy:
    1. In Edit Mode [tab] select the scalp faces.
      Hit Mesh > Separate > Selection.

    2. In Object mode [tab] select the scalp mesh.

      In the Outliner panel rename it "scalp".

    3. Duplicate the scalp mesh [Shift] + [d].

    4. Hide one scalp (disable the eyeball).

    5. Select body and visible scalp, re-combine: Object > Join.

    6. In Edit mode [tab] select around all head vertices.
      Mesh > Merge > By Distance, to merge the pairs of vertices along the seam.

    7. In Outliner, rename the mesh "body", unhide remaining scalp.

    8. Duplicate the scalp, and set Scale X = -1 to make it a mirror.

    9. Combine the two scalp halves: Object > Join.

    10. In Edit Mode [tab] select around all scalp vertices.
      Mesh > Merge > By Distance, to merge the pairs of vertices along the seam.


    b. Smooth and fix normals:
    1. Hit [tab] for object mode.
    2. Object > Shade Smooth.
      Any strange dark shading is a sign of bad "Normals" (some polygons
      facing the wrong direction. All polygons should face outward).
    3. To fix reversed Normals: Mesh > Normals > Recalculate Outside.



    c. Extrude and shape the hair (liked mashed potatos):
       
    1. In Edit Mode [tab], Face mode [3], select around all Faces of the scalp.
       
    2. Hit [e] to Extrude, pull out only slightly, then [s] to scale slightly larger. The hair now is 3-dimensional.
       
    3. Hide the body mesh. Orbit to view under the hair.
       
    4. Delete under the hair: Select the original scalp faces
          (select center faces and [Ctrl]+[numpad +] to grow the selection).
          Hit [delete] or [x] and choose "delete faces".

    [6] ANIMATE: Animate the character!
    For games, each animation needs to be a looping clip.
    1. The clips can all be made on a single timeline (with original bind pose keyframe = 0).
       
    2. A basic move set includes an idle (10-70), walk (100-130), run (200-220), jump (300-330), attack (400-430), get hurt (500-530), and knocked out / dead (600-630).

      Blender Rigging Concepts:

      [6.1] What is a rig?

    We create an armature of bones inside the mesh to drive animation. Most bones are connected to a "parent" bone in a hierarchy to inherit transforms (movement or rotation). The type of inheritance i the kinematics.

      [6.2] Forward Kinematics (FK): Parent Controls Child

    By default, bones are animated as in stop motion: rotate a shoulder, then rotate the elbow, then the wrist, down the chain. This can result in nice arcs for organic motion, but is harder for walking or touching objects that require us to put a foot or hand in a particular place.


      [6.3] Inverse Kinematics (IK): Child Controls Parent

    We can create an IK solver from wrist-to-shoulder or foot-to-hip that allows us to move the wrist or foot and "solve" the motion of the elbow or knee in between.


      [6.4] Switch Between Modes: Object, Edit, and Pose:

    • [Object Mode]: The default, used for moving the un-keyframed rig.
    • [Edit Mode]: With rig selected, hit [tab] to enter / exit [Edit Mode]. Used for creating new bones ([e] to extrude) and transforming existing bones (position [g], rotate [r]).
    • [Pose Mode]: With rig selected, hit [Ctrl/Cmd]+[tab] to enter / exit [Pose Mode]. Used for animating bones and assigning Constraints (like Inverse Kinematics).


      [6.5] Disconect or Connect a Bone:

    • To separate a bone: Extrude it out of an existing bone and hit [Alt/Option]+[p] and choose "clear parent" to unparent it.
    • To parent a bone to another bone: Select the intended child, then the intended parent, hit [Ctrl]+[p].

      [6.6] Connect Mesh to a joint Rig:

    Select all mesh parts first, then select the rig root control last. Hit [Ctrl/Cmd]+[ p ], choose "Automatic Weights" from the tool menu.

    In the top left, switch from Object Mode to Pose Mode for rig controls and posing.

               and or Location key (note the RotLoc option).

    4. Set changed pose keyframes at frame 40:
          a. Select a bone you want to change.
          b. Set the Rotation and / or Position changes ion the 3D View.

               and or Location key (note the RotLoc option).

    PlayerMove3D.cs:
    using System.Collections;
    using System.Collections.Generic;
    using UnityEngine;

    public class PlayerMove3D : MonoBehaviour {
        Animator anim;
        Rigidbody rb;
        public Transform cam;

        public float speed = 6f;
        public float turnSmoothTime =0.1f;
        private float turnSmoothVelocity;

        void Start(){
            anim = gameObject.GetComponentInChildren<Animator>();
            rb = gameObject.GetComponent<Rigidbody>();
            cam = GameObject.FindWithTag("MainCamera").GetComponent<Transform>();
        }

        void Update () {
            float horiz = Input.GetAxisRaw("Horizontal");
            float vert = Input.GetAxisRaw("Vertical");
            Vector3 direct = new Vector3(horiz, 0f, vert).normalized;

            if (direct.magnitude >= 0.1f) {
                float targetAngle = Mathf.Atan2(direct.x, direct.z) * Mathf.Rad2Deg + cam.eulerAngles.y;
                float angle = Mathf.SmoothDampAngle(transform.eulerAngles.y, targetAngle, ref turnSmoothVelocity, turnSmoothTime);

                Vector3 moveDir = Quaternion.Euler(0f, targetAngle, 0f) * Vector3.forward;
                rb.MovePosition(transform.position + moveDir * speed * Time.deltaTime);
                anim.SetBool("walk", true);
            } else {
                anim.SetBool("walk", false);
            }
        }
    }

    LESSON #9: Creating Objects for Export to Unity

    3D Assets for an interactive / game environment should:

    • Be sized to the game units
    • Keep within poly-budget limits
    • Include UV Unwrap and textures.
    • Be located at world center for the export.

      [9.1] SIZING:

    The Unity game engine and Blender both use Meters by default.

    Character Proxy:
    To size your objects for Unity, create a a cube the size of a person in Unity, to base the sizing of all other objects.
    Dimensions: 2 meters tall (z), 1m wide (x), 1m deep (y):
    1. Add menu > Mesh > Cube.
    2. Open the LowerLeft tool panel.
      Note default size = 2m.
      This is the correct height (z) for a Unity character. We want to change the width (x) and depth (y).
    3. Hit [n] to open Transforms.
      Set Scale X and Y = 0.5 (which will be 1 meter, each).
    4. Move the proxy up to stand on the ground:
      Near top of Transforms, set Location > Z = 1.


    How to move the object Pivot, to scale up from the ground:
    1. Select mesh, hit [tab] to enter [Edit Mode].
    2. Orbit [MiddleMouse ClickDrag] to look under the object.
    3. Choose Face mode [3].
    4. Select the bottom Face.
    5. Move the Cursor: Mesh > Snap > "Cursor to Selected".
    6. Hit [tab] for [Object Mode].
    7. Move the Pivot: Object > Set Origin > "Origin to 3D Cursor".
    8. Reset the cursor: [Shift]+[s] > "Cursor to World Origin"
      You can now Scale [s] the object and it will stay grounded.


      [9.2] POLY BUDGET:

    A model's triangle budget for a game depends on its use. If the model is only meant to be used once, like a fountain in a town square, it can be a bit higher-poly. If it is a tree intended to fill a forest with dozens of copies, each tree needs to be as low-poly as possible.

    The triangle poly count of a mesh is a key game performance issue, along with texture size, object count, and (ideally) having no interior geometry.

    Check Polygon Count:
    1. In Viewport UpperRight Corner, find Viewport Overlays tab (rolldown next to X-Ray).
    2. Open it and turn on "Statistics".
    3. Select an object to see its triangle count.


    To remove unneeded vertices:
    1. Select two or more wasted vertices on a straight line
    2. Hit [m] Merge > At Last / At First to target weld


    USEFUL SHORTCUTS:
    1. [L] = select area (works for Faces in 3D View and UV View)
    2. [a] = select all.
    3. [i] = keyframe (use hovering over Transforms [n], choose "Available")

    2. In the Inspector, Animation tab, hit play to see all animations
    3. Create 3 new clips, with the following names, ranges, and loop status:

      "player_walk"
      "player_jump"       		10-70

      200-220       		loop = on
      loop = on
      loop = off
    4. Be sure to hit [apply] for each.


    b. Make the Player object
    1. In the Hierarchy, create a new Empty GO.
    2. In Inspector: Reset Transforms, name it Player, and add tag "Player"
    3. [Add Components]:
          Physics > BoxCollider

    c. Add the FBX mesh to the Scene
    1. Drag the FBX from Project panel into Hierarchy.
    2. Drag it onto the Player object it make it a child.
    3. Name it "player_art".


    d. RightClick the Assets folder to create a new Animator Controller.
    1. Name it "Player_Animator".
    2. Drag it onto player_art to add it as a Component.
    3. Open the triangle next to the FBX in the Assets folder to see the clips (triangles).
    4. DoubleClick the animator to open the Mecanim state machine panel, and drag all three clips into it.


    e. Create Transitions:
    1. RightClick a State to add a Transition, then click on the destination.
    2. Make sure [player_idle] is the Default State (should be orange-- if it is not, right-click and set to Default).
    3. Add Transitions to and from [player_idle] and [player_walk].
    4. Create a transition from [Any State] to [player_jump], and from [player_jump] to [player_idle].
    5. In Parameters (left), create a bool named "walk" and a trigger named "jump".

    f. Add a PlayerMove script
    1. RightClick the Project panel Assets folder to create a new C# script.
    2. Name it "PlayerMove.cs".
    3. Open in your editor and copy the content below into it.
    4. Save the script, return to Unity to compile, and add it to your Player parent object.

    PlayerMove.cs

    using UnityEngine;
    using System.Collections;

    public class PlayerMove : MonoBehaviour {
          public float speed = 100f;
          Animator anim;
          Rigidbody rigidbody;
          public float JumpSpeed = 25f;

          void Start(){
                anim = gameObject.GetComponentInChildren<Animator>();
                rigidbody =gameObject.GetComponent<Rigidbody>();
          }

          void FixedUpdate(){
                if (Input.GetAxis("Vertical") > 0) {
                      rigidbody.velocity += transform.forward * speed;
                      anim.SetBool ("walk", true);
                } else if (Input.GetAxis("Vertical") < 0) {
                      rigidbody.velocity += transform.forward * -1 * speed;
                      anim.SetBool ("walk",true);
                } else if (Input.GetAxis("Horizontal") > 0) {
                      rigidbody.velocity += transform.right * -1 * speed;
                      anim.SetBool ("walk",true);
                } else if (Input.GetAxis("Horizontal") < 0) {
                      rigidbody.velocity += transform.right * speed;
                      anim.SetBool ("walk",true);
                } else if (Input.GetKey("space")){
                      anim.SetTrigger("jump");
                      rigidbody.AddForce (Vector3.up * JumpSpeed);
                } else {
                      anim.SetBool ("walk", false);
                      rigidbody.velocity = new Vector3(0,rigidbody.velocity.y,0);
                }
          }
    }

    2. Lift it a bit above the ground box.
    3. Hit [Play]!
