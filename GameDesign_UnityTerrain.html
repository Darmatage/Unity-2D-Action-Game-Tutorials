[STEP 1]: CREATE THE TERRAIN OBJECT:
In Unity create a Terrain object: GameObject > 3D Objects > Terrain



[STEP 2]: SET TERRAIN SIZE:
Set the initial size and resolution of the Terrain, to get the desired level of scultping detail.


In Hierarchy select theTerrain. In the Inspector we see the controls and settings for the Terrain.
Click the far right button (#5). It displays a little Gear icon over mountains, for main settings.
Under Resolution: set Terrain Width and Length = 100
(so we pack the same amount of detail into a much smaller space).



[STEP 3]: SET SCENE LIGHTING:
Adjust the lighting to make it easier to see your Terrain sculpting.
In the Hierarchy, select the default Directional Light.
In the Inspector change the color to a slightlydarker yellow or blue, so the Terrain is easier to see.



OVERVIEW:
The first four buttons in the Terrain Inspector settings are for manipulating your Terrain.
The first button allows us to add more terrain patches connected to the first.
The second button includes all of the sculpting and painting options (see steps 4 an 5, below).
The third and fourth are for tree and foliage placement (see step 6, below).




Here are the options under the second button, for sculpting and painting, covered in the tutorials below (click each to see full image):


 
[STEP 4]: SCULPTING:
With the Terrain object selected in the Hierarchy:
a. Sculpt the Hills and Valleys:
Use "Raise or Lower Terrain" to sculpt hills.
In the Inspector: click the 2nd Terrain button.
Click the rolldown to choose "Raise or Lower Terrain".
Under "Brushes" set Size = 30 and Opacity = 3 (way down, for much slower changes).

In the Scene view, Zoom out to see more of your Terrain [Alt] + RightMouse-Click-drag.
LeftClick-drag (draw) on the selected Terrain in the Scene view to raise geometry.
To lower the geometry (valleys) hold [Shift]+LeftClick-drag.

In the Inspector, adjust the Opacity parameter for the selected tool for slower or faster changes.
Change your Brush Shape or Size to draw with other patterns and create rougher areas.


b. Sculpt the Flats:
Use "Set Height" to create flat areas (better for gameplay spaces):
In the Inspector, click the rolldown to choose "Set Height" and see its parameters.

Move mouse to desired height. Hold [Shift] to choose it (see new height value set in Inspector).
Draw (LeftClick-drag) your flattened plateau.
Alternatively, just set the Height in the Inspector settings for the Flatten tool.

NOTE: To Sculpt valleys or rivers lower than the default height, change the default height at start using the Flatten tool: set Height = 50, hit [Flatten]. Then start sculpting up and down from there!


c. Sculpt the Smoothness:
Use Smooth Height to smooth any rough cliff edges.
In the Inspector, click the rolldown to choose "Smooth Height" and see its parameters.
Increase Brush Size and Opacity.
LeftClick-drag on rough areas of the Terrain in the Scene to smooth them out.
Increase Opacity significantly to see faster results.



Two sculpting tools not covered in this tutorial are Paint Holes and Stamp Terrain.

PAINT HOLES allows us to draw hole sin the Terrain. This is ideal for adding buildings to the Scene with levels below the Terrain level, so players can move down into the building without being blocked by the terrain collider.

STAMP TERRAIN allows us to add more granular, 3D sculpted surface detail from an image.


 
[STEP 5]: TEXTURES:
Use Paint Texture to add layers of tiling color and bump maps to the Terrain surface.

a. Add Texture PNGs to your project:
If you have not already done so, please download these 3 simple tiling PNGs (click each, save to your desktop).
Import all 3 images into the Unity Project panel (drag in, or RightClick > Import New Asset).


b. Open Paint Textures:
Re-select the Terrain in the Hierarchy.
In the Inspector, click the rolldown to choose "Paint Texture" and see its parameters.

When we start, we have no textures to paint. We need to add them.
Hit the [Edit Terrain Layers] button.
Choose the "Add texture" option.

c. Add the first Texture PNG:

In the "Select Texture2D" wndow that opens, type in the top search bar "tex" to more easily see our three imported images.

Select the first image (green tex1_grass).

Close this window, and the new texture layer is automatically applied to the entire Terrain.



d. Add the other two Texture layers:
Repeat for the earth and stone textures. For each:
Hit [Edit Terrain Layer] button.
Choose Create Layer.
Select the image from the "Select Texture2D window.


e. Paint the other Textures to create a more complex Terrain surface:
Select the 2nd or 3rd texture layer in the Inspector.
LeftClick-drag on the Terrain, to paint over the first texture.
Reduce Brush Size to draw smaller lines, and to create blending set Opacity low. Also try other brush shapes.

To "erase", simply draw over the areas you want to change (like the grass over stone walkway in the image below).

STYLE SUGGESTIONS: Try to use Texture to both:
    (a) emphasize your sculpted forms (hill sides a texture, tops another, valleys/ rivers another, etc)
    (b) spotlight gameplay relevant spaces (ancient stone path, old stone or wood foundation, etc)


f. Change Texture Tiling Size:
Select a texture layer. Click the triangle below to open the layer rolldown and view its properties.
Towards the bottom find Tiling Settings and adjust the X and Y Tiling to make the texture details bigger or smaller (decrease from 2 to 0.5 to make the details smaller).
Note the other cool properties that can be adjusted to get more detailed or realistic textures, including adding a bump map (drag a bump into the Normal Map slot) and adjusting specularity.

 
[STEP 6]: Add trees or other foliage / rocks.
The third button is for placing "Trees" -- repeated 3D assets that automatically hug the Terrain slopes.
a. Add Tree objects to your Project panel:
Download this TREES Unity Package and save it to your desktop.
Import the package into Unity:
    In a Project panel empty area, RightClick > Import Package and click Custom Package.
    Select TerrainTrees. The Package opens to show the contents. Hit [Import] to Import all.

Now added to your project: a PNG, Unity Material, tree FBX, and a Prefab called TreeA_prefab.


b. Add a Tree to the Terrain options: Open [Edit Trees]
Re-select the Terrain in the Hierarchy.
In the Inspector, select the third Terrain tool button to see Tree parameters.
Hit the [Edit Trees] button.
c. Add a Tree to the Terrain options: Add a Tree Prefab
In the "Add Tree" window that opens, click the hotspot.
In the "Select GameObject" window that opens, choose TreeA_Prefab.
Back in the "Add Tree" window, the Prefab is loaded. Hit [Add] to confirm.


d. Set theTree Parameters, and Paint Tree Objects onto the Terrain:
In the Inspector, under Settings, reduce Brush Size, and increase Density,
Extend the range of Tree Height for more variety (set the range to the left for smaller trees overall).
In the Scene, LeftClick-drag on the Terrain to draw trees.
Hold [Shift] and click on a tree to remove it.

NOTE: This system is more performant than simply dragging the meshes into the Hierarchy and placing them manually, but don't overdo it with hundreds of trees, or expect slow-down or crashes.

 
[STEP 7]: Add an FPS Controller, and Play!
Add a First Person Controller prefab:
Click this link to download our basic FPS Unity Package and save it to your desktop.
In the Unity project panel, rightclick to choose Import Package, select this package.
Find the FPS Prefab here:
Drag it into the Scene where you want to play, lift it above the Terrain in that area.
(so the player does not start below ground and fall).

Hit [Play] to walk around your new Terrain
(WASD to move, Space to Jump, camera controled by mouse).


Multiple Terrain Objects:
PLEASE NOTE: You are not limited to a single terrain object.
A cave is best made with at least two terrains, one flipped upside down to point downwards.
Larger project frequently use multiple terrains for performance, so the entire landscape does not need to be loaded at once, or processed in every frame. Just be mindful of the seams between landscapes: for polished projects, strive to pyut elements there to hide the seam (architecture, fallen logs, bushes, etc).

MOVING SURFACE TEXTURE (for water, lava, etc)

1. MATERIAL: RightClick the Project panel to Create > Material.
Apply your tiling 2D texture art to the Material "Albedo" slot.
Consider making a Normal map and Height map using CrazyBump, to applying those also for greater depth.
Drag the material onto an object in the Hierarchy (the mesh that is meant to be water, lava, etc)

2. SCRIPT: Create a new C# script, name it "MoveTexture.cs".
Apply to the 3D object in the Hierarchy with the Material you want to move.
MoveTexture.cs (for a single animation parameter, "moving"):
using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class MoveTexture : MonoBehaviour {

       public float scrollSpeed = 0.1f;
       Renderer rend;

       void Start () {
              rend = GetComponent<Renderer>();
       }

       void Update() {
              float moveThis = Time.time * scrollSpeed;
              rend.material.SetTextureOffset("_MainTex", new Vector2(0, moveThis));
              //swap x and y to move in the other axis: (moveThis, 0)
       }

}

Try experimenting with this color and this specular / bump map.



 
FLICKERING LIGHTS
Create a new C# script, name it "FlickeringLights.cs".
Apply to the light you want to flicker (or allow it to generate its own light source):
FlickeringLights.cs (source) :
using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class FlickeringLights : MonoBehaviour {

       public new Light lightf; // set this to null if you want the script to be applied to a light in your Scene
       private float minIntensity = 0f;
       private float maxIntensity =1f;
       private static float sharedFlicker;
       [Range (1, 50)]
       public int smoothing =5;
       Queue<float> smoothQueue;
       float lastSum = 0;

       public void Reset () {
              //Reset randomness and start again
              smoothQueue.Clear();
              rend = GetComponentInChildren<Renderer> ();
              lastSum = 0;
       }

     void Start () {
              smoothQueue = new Queue<float(smoothing);
              rend = GetComponentInChildren<Renderer> ();
              if (light == null) {
                     light = GetComponent<Light>();
              }
       }

       void Update() {
              if (light == null){ return;}

              while (smoothQueue.Count >= smoothing){
                     lastSum -= smoothQueue.Dequeue();
              }

              float newVal = Random.Range(minIntensity, maxIntensity);
              smoothQueue.Enqueue(newVal);
              lastSum += newVal;

              light.intensity = lastSum / (float)smoothQueue.Count;
       }
 
}
