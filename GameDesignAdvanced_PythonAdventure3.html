TEXT ADVENTURE #5: Including Multimedia with PyGame
INSTALL PYGAME
Go to PyGame.org, see information here:
https://www.pygame.org/wiki/GettingStarted#Pygame%20Installation
We use PIP (python package installer) to auto-install libraries and modules. In the regular cmd prompt:
> python -m pip install -U pygame --user
See if it works, run an included game example in the cmd prompt:
> python -m pygame.examples.aliens

NOTE: If this fails, we likely did not connect the original PATH when installing ("Add python 3.6 to PATH"), and need to establish that path.
WINDOWS 10: RightClick "This PC", choose Properties, choose "Advanced System Settings," choose Environmental Variables. Select "Path", hit [Edit], and add paths with [New]. Like:
C:\Users\darmatage\AppData\Local\Programs\Python\Python38-32
C:\Users\darmatage\AppData\Local\Programs\Python\Python38-32\Scripts
(NOTE: find the path by going to IDLE and hitting File > SaveAs)
Test in cmd prompt:
> python --version and pip --version.

...OR simply cd your way to the pythonVersion folder, and run the install command there
> cd AppData\Local\Programs\Python\Python38-32\
> python -m pip install -U pygame --user

restart IDLE, and test by typing in the shell prompt
>>> import pygame

Images (PNG, GIF, or JPG), audio files (WAV or MP3), and font files (TTS) should be located in the same folder as the .py file/s that reference them. PNGs are well suited for images with transparency, to avoid default hard-rectangle edges.
